# BigBlueButton API for PHP

The **BigBlueButton API for PHP** makes it easy for developers to access [BigBlueButton][bbb] API.
API in their PHP code.

## Development

Running PHPCS-Fixer

```
./vendor/bin/php-cs-fixer fix
```

## Runing tests

```
./vendor/bin/phpunit
```

[bbb]: http://bigbluebutton.org
